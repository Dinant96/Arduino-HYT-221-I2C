Arduino-HYT-221-I2C
===================

Output values read from the HYT family via I2C on the Arduino serial console.

Pinout:
HYT 221 view from top (Should be the same for all members of the HYT family. Check the Datasheet though.)

     ________
    | =    = |
    |   __   |
    |  /  \  |
    |  \__/  |
    |        |
    |_|_|_|_||
      | | | |
      | | | |
      | | | |
     SDA|VDD|
        |   |
       GND SCL
